# Triangle Simulator

## Testing & Linter Directory layout

There are one components to test


To run the unit test, you can execute.

```
yarn run test
```

To run the linter, you can execute.

```
yarn run lint
```

## How to

### Setup when development

```
make setup-dev
```

### Run web development

To run a web using `yarn`

```
yarn run dev
```


### Build asset and run production

To build new asset using `yarn`

```
yarn start
```
